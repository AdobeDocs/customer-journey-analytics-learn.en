# Validate data and identity stitching quality using preview

Learn about the dataset preview feature in Customer Journey Analytics that delivers instant identity stitching metrics—including Person ID coverage and Persistent ID coverage—directly when configuring connections.

Evaluate dataset suitability for reliable cross-device and cross-channel reporting before ingestion, dramatically reducing time to value and de-risking identity strategy decisions with real-time insights.

For more information, review the [documentation](https://experienceleague.adobe.com/en/docs/analytics-platform/using/stitching/validate){target="_blank"}.
