# Add [!UICONTROL area visualizations] to [!DNL Analysis Workspace] projects in [!DNL Customer Journey Analytics]

Learn how to add [!UICONTROL area] and [!UICONTROL area stacked] visualizations to [!DNL Analysis Workspace] projects in [!DNL Customer Journey Analytics]. These visualizations are ideal to use when there are multiple metrics for which you want to visualize the area based on the intersection.

>[!VIDEO](https://video.tv.adobe.com/v/3479637/?quality=12&learn=on)

For more information about [!UICONTROL area] visualizations, visit the [documentation](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-workspace/visualizations/area.html){target="_blank"}.
