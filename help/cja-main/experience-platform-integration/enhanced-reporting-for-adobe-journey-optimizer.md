# Enhanced reporting for Adobe Journey Optimizer

New reporting and orchestration measurement features are available in Adobe Journey Optimizer. These enhancements to Journey Optimizer also include increased interoperability and extended value with Customer Journey Analytics through additional data sources, advanced customization, data manipulation, analysis, and more. For detailed documentation, see [Get started with new Reporting interface](https://experienceleague.adobe.com/en/docs/journey-optimizer/using/channel-report/report-gs-cja){target="_blank"}.
