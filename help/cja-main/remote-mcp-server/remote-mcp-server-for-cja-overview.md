# Remote MCP server for [!DNL Customer Journey Analytics] overview

Learn how to connect AI tools like [!DNL Claude], [!DNL ChatGPT], or [!DNL Cursor] to the Adobe Customer Journey Analytics (CJA) MCP server to explore data views, execute reports, and generate executive-ready KPI insights using natural language. This demo showcases how Remote MCP servers enable scalable, governed analytics workflows and automated reporting in Adobe Experience Platform.

For more information, review the Content Analytics [documentation](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"}.
